**A. Create `packages/llm/src/prompts/diagnose-probe.ts`:**

Export `buildProbeSystemPrompt(probeQuestion: string, gradeLevel: number, originalQuestion: string): string`.

System prompt content per research:
- Role: Socratic science tutor for grade band (use same K-5/6-8/9-12 logic as enrich.ts)
- Goal: understand what student genuinely believes, do not correct yet
- Include the probe question from library as starting point, instruct LLM to adapt wording to sound natural
- Rules: one question only, no hints at correct answer, never use "misconception"/"wrong"/"incorrect", no empty praise, end with exactly one question mark
- Include original student question for context

**B. Create `packages/llm/src/prompts/diagnose-confront.ts`:**

Export `buildConfrontSystemPrompt(misconceptionName: string, studentProbeResponse: string, confrontationSeed: string, gradeLevel: number): string`.

System prompt content per research:
- Role: Socratic science tutor, student has shared their mental model
- Present real-world counterexample that creates cognitive conflict
- Include misconception name for LLM context (NOT in student-visible output — the system prompt instructs LLM to never mention the misconception name)
- Include student's probe response so confrontation is personalized
- Use confrontation_scenarios[0] as seed, adapted to student's words and grade level
- Do not tell student they are wrong; present evidence and ask them to reconcile
- End with a question

**C. Create `packages/llm/src/prompts/diagnose-resolve.ts`:**

Two exports:

1. `evaluateResolution(params: { misconceptionName: string; probeResponse: string; confrontationUsed: string; studentFinalResponse: string }): Promise<{ resolved: boolean; rationale: string }>` — Uses `generateText + Output.object` pattern from extract.ts. Zod schema: `z.object({ resolved: z.boolean(), rationale: z.string() })`. Evaluation prompt per research: generous — partial understanding counts as resolved; only return false if student completely restates original misconception or shows no engagement.

2. `buildResolveMessage(misconceptionName: string, resolved: boolean): string` — Returns system prompt for the final reveal message. If resolved: affirm thinking shift warmly. If unresolved: be kind, encourage revisit. Always reveal: "What you were exploring is actually a named scientific misconception called '[misconceptionName]'. Many curious minds have thought the same thing." Keep under 150 words, do not be preachy.
