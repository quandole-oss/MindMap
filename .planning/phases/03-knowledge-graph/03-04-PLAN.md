  <action>
1. Create `apps/web/lib/graph/centrality.ts`:
   - Export function `computeBetweennessCentrality(nodeIds: string[], edges: Array<{ source: string; target: string }>): Map<string, number>`:
     - Build adjacency list: `Map<string, string[]>`
     - For each edge, add both directions (undirected)
     - Implement Brandes' algorithm (BFS from each source node):
       - Initialize stack, predecessors map, sigma (shortest path counts), distance map, delta (dependency)
       - BFS from source: track predecessors and sigma for shortest paths
       - Back-propagation: compute betweenness dependency delta
       - Accumulate centrality for each node (divide by 2 for undirected)
     - Return Map<string, number> of node ID -> centrality score
   - Export function `findTopBridgeNode(nodes: Array<{ id: string; domain: string }>, edges: Array<{ source: string; target: string }>): { nodeId: string; connectedDomains: [string, string] } | null`:
     - If fewer than 3 nodes or 2 edges, return null (no meaningful bridge)
     - Compute betweenness centrality via `computeBetweennessCentrality()`
     - For each node, determine the set of domains it bridges (look at its neighbors' domains; a bridge connects nodes from at least 2 different domains)
     - Filter to nodes that actually connect different domains (not just high centrality within one domain)
     - Return the node with highest centrality that bridges at least 2 domains, along with the two most prominent domain names it connects
     - If no cross-domain bridge exists, return null

2. Update `apps/web/actions/graph.ts`:
   - Add import: `import { findTopBridgeNode } from "@/lib/graph/centrality"`
   - In `getGraphData()`, after fetching nodes and edges:
     - Call `const bridge = findTopBridgeNode(nodes, edges.map(e => ({ source: e.source, target: e.target })))`
     - If bridge is not null, mark the bridge node: set `isBridge: true` for the node with matching ID
     - All other nodes keep `isBridge: false`
   - Export new async function `getBridgeConnection(): Promise<{ bridgeNodeId: string; bridgeNodeName: string; domainA: string; domainB: string } | null>`:
     - Get session, fetch graph data (reuse internal query logic)
     - Call `findTopBridgeNode()`
     - If result exists, look up the bridge node name
     - Return bridge info or null if no bridge exists
  </action>
